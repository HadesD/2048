# 2048
Game 2048 - C++ version

## Play

- Up: <kbd>k</kbd> or <kbd>w</kbd>
- Down: <kbd>j</kbd> or <kbd>s</kbd>
- Left: <kbd>h</kbd> or <kbd>a</kbd>
- Right: <kbd>l</kbd> or <kbd>d</kbd>
